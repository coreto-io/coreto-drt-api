## Requirements

- [Node.js v16.15.0](https://nodejs.org/en/download/package-manager/)
- [npm v8.5.5](https://www.npmjs.com/get-npm)

---

## Setup

1. Clone repository

2. Install dependencies

```bash
npm install
```

3. Configure `.env` from `.env.example`

4. Start server

```bash
npm start
```
